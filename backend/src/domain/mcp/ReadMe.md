# MCP Domain

## Overview

The MCP Domain is responsible for managing MCPs for the application.

## Features

- Create MCPs
- Update MCPs
- Delete MCPs
- Get MCPs

## Components

```bash
domain/mcp/
├── MCP.ts             # Entity (data structure)
├── MCPRepository.ts   # Interface for saving & fetching
├── MCPService.ts      # Core logic (create, update, delete, get)
```
